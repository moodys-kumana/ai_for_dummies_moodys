üìö Glossary of Key AI & Programming Terms

ü§ñ AI Concepts

Artificial Intelligence (AI)

Computer systems that perform tasks requiring human intelligence

Machine Learning

AI that learns from data to make predictions and decisions

Deep Learning

Advanced AI mimicking human brain neural networks

Neural Networks

AI systems inspired by biological neural networks

Natural Language Processing (NLP)

AI that understands and generates human language

Computer Vision

AI that interprets and analyzes visual information

Chatbots

AI programs that simulate human conversation

Generative AI

AI that creates new content like text, images, or code

üíª Programming Concepts

Programming/Coding

Writing instructions for computers in special languages

Programming Languages

Formal languages with specific syntax for coding (Python, Java, etc.)

Scripts

Small programs that automate specific tasks

Libraries & Frameworks

Pre-written code packages that simplify development

APIs

Application Programming Interfaces - ways for programs to communicate

Databases & SQL

Organized data storage and the language to query it

Front-end/Back-end

User interface vs. server-side logic in applications

Open Source

Software with freely available source code

üìä 2. Data Analysis and AI for Insights
